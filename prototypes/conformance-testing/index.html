WCAG 3 Conformance Testing

There are currently four sites are available that can be used to test WCAG 3 conformance:

  1. Accessible Escape Room
  2. A login form
  3. The GOV UK suite of pages originally created to test automated testing tools.
  4. The APCA (Advanced Perceptual Contrast Algorithm) visual contrast testing tool.

Accessible Escape Room

The Accessible Escape Room is a customized version of an existing site (reproduced with permission). The site has been altered to introduce content that falls under the scope of the the First Public Working Draft of WCAG 3.

Login Form

The login form has a number of pages that appear in a common user journey to to log into, or register with, a site. Note that none of the pages submit any kind of data.

GOV.UK Test Suite

These pages, reproduced under the MIT licence (our copy of which is available in this site's repo), were originally created to test various accessibility scanning tools. This GOV.UK Testing For Accessibility article has more information on that work.

APCA (Advanced Perceptual Contrast Algorithm) visual contrast testing tool

This is a proof of concept prototype to test the new color contrast algorithm.
