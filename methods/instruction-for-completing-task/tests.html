Atomic Tests

  • Test Units: Each input.
  • Unit Score: For each test unit, complete each of the following tests and sum the test credits to get the unit score.

Test that instructions are present

Test Procedure

  1. Identify inputs that only accept data in a given format.
  2. Check that the page or view provides details about the required format.

Expected Results

#2 is true.

Critical Error if #2 is false.

Test for clear words

Test Procedure

  1. Identify instructions.
  2. Check that the instructions use clear words.

Expected Results

#2 is true.

Test that illustrative examples are present

Test Procedure

  1. Identify instructions that would benefit from an illustrative example.
  2. Check that the instructions include an example of data in the required format.

Expected Results

#2 is true.

Exception: Password inputs due to other considerations, for example, security risk
