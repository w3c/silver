Atomic Tests

  • Test Units: Each input that is required.
  • Unit Score: For each test unit, complete each of the following tests and sum the test credits to get the unit score.

Test that required inputs are visually identifiable

Test Procedure

  1. Check that the required status is conveyed visually.
  2. For non-text visual indicators:
    1. Check that instructions are provided that explain the non-text indicator.
    2. Check that the non-text indicator has sufficient color contrast.
    3. Check that the non-text indicator is identifiable without color.

Test that required inputs are programmatically defined

Test Procedure

  1. Check that required inputs include the aria-required attribute.
  2. Check that the value of the aria-required attribute is "true".
