Atomic Tests

  • Test Units: Each input that is required.
  • Unit Score: For each test unit, complete each of the following tests and sum the test credits to get the unit score.

Test that required inputs are visually identifiable

Test Procedure

  1. Check that the required status is conveyed visually.
  2. For non-text visual indicators, check that instructions are provided that explain the non-text indicator.

Test that required inputs are programmatically defined

Test Procedure

  1. Check that required inputs include the required attribute and/or the aria-required attribute.
  2. For the aria-required attribute, check that the attribute value is "true".

Expected Results

All checks are true.
