Input instructions provided

Provides instructions for inputs that have requirements (for example, required, date) so users know how to provide valid information.

This outcome relates to guideline Error Prevention.

Methods:

  1. Instructions for completing tasks
  2. Required inputs indicated
  3. Instructions available at the source of input
