Welcome to the home page for the Silver XR Subgroup. We meet as a group weekly (Mondays) and focus on the development of material relating to XR accessibility. Our current focus is on the creation of a draft guideline for WCAG3 that focuses on captioning in XR. The development of this guidance follows a path of activities that is outlined below:

  1. Identify User Needs
  2. Write Functional Outcomes
  3. Write the Tests
  4. Write Methods to Pass Tests
  5. Write how to and supporting documentation
  6. Write the Guidelines

Identify User Needs

We have carried out a task that has taken XR Accessibility User Requirements and identified those that are relevant to captioning. Information on this can be seen on the Captioning XR User Requirements page.
